# Public Assets

Эта папка содержит статические файлы, доступные напрямую через URL.

## Логотип

Поместите ваш файл логотипа сюда и назовите его `logo.png`.

Поддерживаемые форматы:
- PNG (рекомендуется)
- SVG
- JPG/JPEG

### Инструкция:

1. Поместите файл логотипа в эту папку (`public/`)
2. Назовите файл `logo.png` (или обновите путь в `app/components/Logo.tsx`)
3. Рекомендуемые размеры:
   - Минимум: 200x75px для лучшего качества
   - Формат: PNG с прозрачным фоном (если нужно)

### Альтернативные форматы:

Если ваш логотип в другом формате (SVG, JPG), обновите путь в `app/components/Logo.tsx`:
- SVG: `/logo.svg`
- JPG: `/logo.jpg`
